# Canvas Sections Tags

This section contains all tags related to the core Social Lean Canvas framework components, organized in a hierarchical structure.

## Structure Overview

### [Purpose](purpose/purpose.md)
Foundational problem statements and venture purpose.

### [Customer Model](customer-model/customer-model.md)
Value creation for customers including:
- [Customers](customer-model/customers/customers.md) (and [Early Adopters](customer-model/customers/early-adopters/early-adopters.md))
- [Jobs to be Done](customer-model/jobs-to-be-done/jobs-to-be-done.md) (and [Existing Alternatives](customer-model/jobs-to-be-done/existing-alternatives/existing-alternatives.md))
- [Unique Value Proposition](customer-model/unique-value-proposition/unique-value-proposition.md)
- [Solution](customer-model/solution/solution.md)

### [Impact Model](impact-model/impact-model.md)
Purpose to outcomes chain including:
- [Issue](impact-model/issue/issue.md)
- [Participants](impact-model/participants/participants.md)
- [Activities](impact-model/activities/activities.md)
- [Outputs](impact-model/outputs/outputs.md)
- [Short-term Outcomes](impact-model/short-term-outcomes/short-term-outcomes.md)
- [Medium-term Outcomes](impact-model/medium-term-outcomes/medium-term-outcomes.md)
- [Long-term Outcomes](impact-model/long-term-outcomes/long-term-outcomes.md)
- [Impact](impact-model/impact/impact.md)

### [Economic Model](economic-model/economic-model.md)
Financial sustainability including:
- [Channels](economic-model/channels/channels.md)
- [Revenue](economic-model/revenue/revenue.md)
- [Costs](economic-model/costs/costs.md)
- [Advantage](economic-model/advantage/advantage.md)
- [Financial Model](economic-model/financial-model/financial-model.md)

### [Key Metrics](key-metrics/key-metrics.md)
Cross-model measurement and tracking.

## Usage Rules

1. **Hierarchical Tagging**: Always include parent tags when using child tags
2. **Required Combinations**: Some tags require specific parent tags
3. **Cross-section Content**: Content can span multiple canvas sections

See individual tag definition files for specific usage criteria and examples.
