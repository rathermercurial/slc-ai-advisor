# issue

## Definition
The `issue` tag identifies content about the social or environmental problem articulated in the impact model - a clear statement of the issue identified in the purpose that the enterprise aims to address.

## Usage Criteria
Apply this tag to content that:
- Defines problem identification and articulation approaches
- Discusses issue framing and scoping methodologies
- Provides frameworks for problem analysis
- Analyzes issue definition in case studies

## Always Include When
- Content focuses on social/environmental problem definition
- Issue identification and scoping frameworks
- Problem framing methodologies
- Root cause analysis approaches

## Never Include When
- General problem-solving content not specific to social/environmental issues
- Purpose statements without specific issue focus
- Customer problems unrelated to social/environmental impact

## Required Combinations
- Must include appropriate canvas-sections hierarchy: [impact-model, issue]

## Content Examples
- Issue identification frameworks
- Problem scoping methodologies
- Root cause analysis techniques
- Issue definition templates and examples

## Related Tags
- Parent tags: [impact-model](../impact-model.md)
- Often appears with purpose (issue reflects purpose)
- Often paired with [participants](../participants/participants.md) (who is affected by the issue)
- May combine with `case-study` content tags for issue analysis examples

## Key Concepts
- Social vs. environmental issues
- Issue scoping and boundaries
- Root cause vs. symptoms
- Issue evidence and data
- Stakeholder perspectives on issues
- Issue urgency and scale
- Systemic vs. specific issues
