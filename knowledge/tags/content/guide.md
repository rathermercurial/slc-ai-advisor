# guide

## Definition
The `guide` tag identifies content that provides step-by-step instructions, how-to documentation, or procedural guidance for completing specific tasks within the SLC framework.

## Usage Criteria
Apply this tag to content that:
- Provides step-by-step instructions
- Offers how-to guidance for specific tasks
- Contains procedural documentation
- Walks users through processes

## Always Include When
- Content has numbered steps or procedures
- How-to instructions are the primary purpose
- Practical implementation guidance is provided

## Never Include When
- Pure conceptual or theoretical content (use [concept](concept.md))
- Reference documentation without actionable steps
- Case studies without procedural guidance

## Content Examples
- Canvas section completion guides
- Implementation how-tos
- Process documentation
- Onboarding materials

## Related Tags
- Often combined with [template](template.md)
- May include canvas section tags for specific guidance
- Related to [business-model-design](business-model-design.md) for design guides
