# Lexicon Entry

This section contains definitions, terminology, and conceptual explanations that form the lexicon of the Social Lean Canvas framework. It serves as the authoritative source for understanding key terms and concepts.

## Purpose

The lexicon ensures:
- **Consistent Terminology**: Shared vocabulary across all framework materials
- **Clear Definitions**: Precise meanings for technical and specialized terms
- **Conceptual Clarity**: Deep understanding of framework foundations
- **Reference Resource**: Easy lookup for users and LLMs

## Structure

Lexicon entries typically include:
- Clear, concise definitions
- Context and background
- Usage examples
- Related concepts and cross-references
- Practical applications

## Usage

The lexicon serves:
- **New Users**: Understanding framework terminology
- **Practitioners**: Reference for precise concept application
- **LLMs**: Consistent interpretation of framework concepts
- **Contributors**: Standardized language for new content

## Content Types

This section includes foundational documents like framework overviews, core concept explanations, and definitional resources that establish the intellectual foundation of the SLC methodology.
