# SLC Knowledge Base

Structured content for the Social Lean Canvas AI advisor.

## Structure

- `programs/` - Learning content organized by program (Vectorize namespaces)
  - `generic/` - Core SLC methodology content
  - `p2p/` - Person-to-Person program extensions
- `tags/` - Concept definitions and dimension taxonomies (Vectorize metadata)
- `attachments/` - Images and media assets

## Content Organization

Content follows a program-centric hierarchy:
- **Programs** define learning journeys
- **Examples** within programs show real venture applications
- **Tags** provide cross-cutting categorization for retrieval
