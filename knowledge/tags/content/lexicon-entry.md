# lexicon-entry

## Definition
The `lexicon-entry` tag identifies content that provides definitions, terminology, and conceptual explanations forming the lexicon of the Social Lean Canvas framework - the authoritative source for understanding key terms and concepts.

## Usage Criteria
Apply this tag to content that:
- Provides clear definitions of framework terms
- Explains key concepts and terminology
- Offers conceptual foundations and explanations
- Serves as reference material for framework understanding

## Always Include When
- Content focuses on defining framework terms and concepts
- Foundational explanations of key terminology
- Reference material for concept understanding
- Definitional content for framework lexicon

## Never Include When
- Procedural or instructional content without definitional focus
- Examples or applications without conceptual explanation
- Content that uses terms without defining them

## Required Combinations
- Often combined with canvas section tags for section-specific definitions
- May include `concept` for theoretical definitional content

## Content Examples
- Framework overview and core definitions
- Key term definitions and explanations
- Conceptual foundation documents
- Terminology reference materials

## Related Tags
- Often appears with canvas section tags for section definitions
- May combine with `concept` for theoretical lexicon content
- Often paired with `design` for framework design definitions

## Quality Criteria
- Clear, precise definitions
- Comprehensive conceptual coverage
- Consistent terminology usage
- Educational reference value
- Authoritative source material
- Cross-referenced concepts
