# prompt

## Definition
The `prompt` tag identifies content that provides structured prompts, instructions, and guidance for LLM interactions with the Social Lean Canvas knowledge base.

## Usage Criteria
Apply this tag to content that:
- Provides structured prompts for LLM interactions
- Offers guidance for AI-assisted canvas creation
- Includes instructions for knowledge base interaction
- Documents prompt design and optimization

## Always Include When
- Content provides LLM prompts and instructions
- AI interaction guidance and frameworks
- Prompt design and optimization documentation
- Knowledge base interaction protocols

## Never Include When
- General instructional content not specific to LLM interaction
- Human-oriented guidance without AI focus
- Content that mentions prompts without providing them

## Required Combinations
- Must include `design`
- Often combined with `admin` for prompt administration

## Content Examples
- LLM interaction prompts and instructions
- AI-assisted canvas creation guidance
- Knowledge base query and interaction protocols
- Prompt optimization and design documentation

## Related Tags
- Parent tag: `design`
- Often appears with `admin` for prompt administration
- May combine with canvas section tags for section-specific prompts

## Content Characteristics
- Structured prompts and instructions
- LLM interaction optimization
- AI-assisted framework application
- Knowledge base integration guidance
- Prompt effectiveness and refinement
