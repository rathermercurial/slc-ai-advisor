# job-solution-fit

## Definition
The `job-solution-fit` tag identifies content about validating that a proposed solution effectively addresses the jobs-to-be-done for target customers.

## Usage Criteria
Apply this tag to content that:
- Defines job-solution fit validation methodologies
- Discusses techniques for confirming solution-problem alignment
- Provides frameworks for solution validation
- Analyzes job-solution fit in case studies

## Always Include When
- Content focuses on validating solution effectiveness
- Solution-problem alignment assessment
- MVP and prototype validation approaches

## Never Include When
- General solution design without validation focus
- Product-market fit content (later stage)
- Job-customer fit content (earlier stage)

## Required Combinations
- Often combined with venture-stage tags (idea-stage, early-stage)
- May include canvas-sections: [customer-model, solution]

## Content Examples
- Job-solution fit validation frameworks
- MVP testing methodologies
- Solution-problem alignment assessment

- Precedes [product-market-fit](product-market-fit.md)
