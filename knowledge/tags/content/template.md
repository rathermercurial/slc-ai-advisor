# template

## Definition
The `template` tag identifies content that provides reusable templates, formats, and structures for creating Social Lean Canvas documents and related materials.

## Usage Criteria
Apply this tag to content that:
- Provides blank or placeholder templates
- Offers structured formats for canvas creation
- Contains reusable document structures
- Includes templated prompts or frameworks

## Always Include When
- Content is designed for reuse and customization
- Blank canvas templates with placeholders
- Structured formats for specific purposes
- Template documentation and usage guides

## Never Include When
- Content contains completed examples (use `canvas-example`)
- Content is purely instructional without reusable elements
- One-off documents not intended for reuse

## Required Combinations
- Often combined with canvas section tags for section-specific templates
- May include venture stage tags for stage-appropriate templates

## Content Examples
- Social Lean Canvas master template
- Impact Model template
- Canvas section templates
- YAML data structure templates
- Visual representation templates

## Related Tags
- Often appears with specific canvas section tags
- May combine with `design` for template design documentation
- Often paired with `lexicon-entry` for template usage definitions

## Template Features
- Placeholder content ({{PLACEHOLDER}} format)
- Structured YAML data for LLM processing
- Consistent formatting and layout
- Clear usage instructions
- Proper front matter structure
- Visual representation guidelines
