# canvas-example

## Definition
The `canvas-example` tag identifies content that provides real-world examples of completed Social Lean Canvases or canvas sections from actual social enterprises.

## Usage Criteria
Apply this tag to content that:
- Shows completed canvas examples from real ventures
- Demonstrates canvas application in practice
- Provides model canvases for learning
- Illustrates good canvas completion practices

## Always Include When
- Content contains actual completed canvases
- Real venture canvas examples and analysis
- Model canvases used for teaching
- Canvas completion demonstrations

## Never Include When
- Content is theoretical or instructional only
- Templates or blank canvas formats (use `template`)
- Case studies without actual canvas examples (use `case-study` only)

## Required Combinations
- Often combined with `case-study` for detailed analysis
- Should include relevant canvas section tags when focusing on specific sections

## Content Examples
- Patagonia Social Lean Canvas example
- Impact model examples from real ventures
- Sector-specific canvas examples
- Stage-specific canvas examples

## Related Tags
- Often paired with `case-study` for comprehensive analysis
- May combine with canvas section tags for section-specific examples
- Often includes venture stage tags for stage-appropriate examples

## Quality Criteria
- Real venture examples (not fictional)
- Complete or substantive canvas sections
- Educational value for framework learning
- Clear, well-formatted presentation
- Attribution to actual ventures
