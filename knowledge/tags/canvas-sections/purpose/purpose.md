# purpose

## Definition
The `purpose` tag identifies content specifically about the Purpose section of the Social Lean Canvas - the foundational statement defining the social or environmental problem the enterprise aims to solve.

## Usage Criteria
Apply this tag to content that:
- Defines what a purpose statement is
- Provides guidance on writing purpose statements
- Analyzes or critiques purpose statements
- Discusses the role of purpose in the framework

## Always Include When
- Content focuses specifically on purpose statement creation
- Examples of well-crafted purpose statements
- Purpose statement evaluation criteria
- Purpose's relationship to other canvas sections

## Never Include When
- Content mentions purpose only in passing
- General impact or mission content not specifically about the canvas purpose section
- Content about organizational purpose outside the canvas context

## Required Combinations
- Must include appropriate canvas-sections hierarchy
- Often combined with `concept` or `lexicon-entry` in content tags

## Content Examples
- "How to write an effective purpose statement"
- Purpose statement templates and examples
- Analysis of purpose statements in case studies
- Purpose section methodology documentation

## Related Tags
- Often appears with impact-model tags (purpose connects to issue)
- May combine with venture-stage tags for stage-specific purpose guidance
- Parent tags: canvas-sections
