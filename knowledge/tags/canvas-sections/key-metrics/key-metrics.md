# key-metrics

## Definition
The `key-metrics` tag identifies content about quantitative measures indicating the current state of the enterprise's customer, economic, and impact models, tracking progress and validation across all canvas sections.

## Usage Criteria
Apply this tag to content that:
- Defines key performance indicators for social enterprises
- Discusses metrics selection and measurement frameworks
- Provides guidance on metric tracking and analysis
- Analyzes metric effectiveness and evolution

## Always Include When
- Content focuses on enterprise measurement and tracking
- KPI development and selection frameworks
- Metrics strategy and implementation
- Cross-model measurement approaches

## Never Include When
- Detailed analytics or data analysis without strategic metric focus
- Operational metrics not tied to canvas model validation
- Technical measurement tools without metric strategy context

## Required Combinations
- Must include appropriate canvas-sections hierarchy: [key-metrics]

## Content Examples
- Key metrics frameworks for social enterprises
- Metric selection and prioritization approaches
- Measurement system design methodologies
- Metric evolution across venture stages

## Related Tags
- Often appears with all model tags (customer-model, impact-model, economic-model)
- May combine with venture-stage tags for stage-appropriate metrics
- Often paired with `experiment-test` content tags for validation metrics

## Key Concepts
- Leading vs. lagging indicators
- Cross-model metric integration
- Metric evolution across stages
- Actionable vs. vanity metrics
- Impact measurement integration
- Customer-economic-impact metric balance
- Evidence-based decision making
