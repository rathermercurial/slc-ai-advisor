# link

## Definition
The `link` tag identifies content that provides external references, connections, and relationships to resources outside the Social Lean Canvas knowledge base.

## Usage Criteria
Apply this tag to content that:
- Provides external references and connections
- Documents relationships to outside resources
- Includes links to external frameworks and methodologies
- Connects framework concepts to external sources

## Always Include When
- Content focuses on external references and connections
- Documentation of outside resource relationships
- Integration with external frameworks and tools
- Cross-references to external sources and materials

## Never Include When
- Internal knowledge base links and references
- Content that mentions external sources without providing connections
- General reference content without specific external focus

## Required Combinations
- May include canvas section tags when links support specific sections
- Often combined with `resource` for external resource connections

## Content Examples
- External framework connections and relationships
- Reference links to outside resources
- Integration documentation with external tools
- Cross-reference materials and connections

## Related Tags
- Often appears with `resource` for external resource links
- May combine with any canvas section tags for section-specific external connections
- Often paired with `concept` for external conceptual connections

## Content Characteristics
- External reference and connection focus
- Integration with outside resources
- Cross-framework relationships
- External validation and support
- Resource accessibility and availability
