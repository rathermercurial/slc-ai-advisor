# case-study

## Definition
The `case-study` tag identifies content that provides in-depth analysis of social enterprises, offering detailed examination of how organizations apply the Social Lean Canvas framework and achieve impact.

## Usage Criteria
Apply this tag to content that:
- Provides comprehensive analysis of real ventures
- Offers detailed implementation insights and lessons learned
- Includes longitudinal or in-depth venture analysis
- Analyzes venture evolution and outcomes over time

## Always Include When
- Content provides detailed venture analysis beyond simple examples
- In-depth implementation stories and lessons learned
- Longitudinal analysis of venture development
- Comprehensive venture evaluation and insights

## Never Include When
- Simple canvas examples without analytical depth (use `canvas-example`)
- Brief mentions or references to ventures
- Theoretical content without specific venture focus

## Required Combinations
- Often combined with `canvas-example` for detailed canvas analysis
- Should include relevant canvas section tags when focusing on specific sections
- Often includes venture stage tags for stage-specific analysis

## Content Examples
- Comprehensive venture analysis and evaluation
- Implementation story with challenges and solutions
- Longitudinal venture development analysis
- Detailed impact and outcome evaluation

## Related Tags
- Often paired with `canvas-example` for canvas-focused case studies
- May combine with any canvas section tags for section-specific analysis
- Often includes venture stage tags for stage-appropriate insights
- May appear with `concept` for theoretical framework application

## Quality Criteria
- Substantial analytical depth and insight
- Real venture focus with authentic details
- Implementation lessons and takeaways
- Balanced analysis including challenges and successes
- Educational value for framework application
