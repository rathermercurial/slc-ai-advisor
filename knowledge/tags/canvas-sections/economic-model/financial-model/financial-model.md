# financial-model

## Definition
The `financial-model` tag identifies content about the overall financial structure combining revenue, costs, and projections to demonstrate economic viability and sustainability over time.

## Usage Criteria
Apply this tag to content that:
- Defines financial modeling frameworks for social enterprises
- Discusses financial planning and projection methodologies
- Provides approaches for financial viability validation
- Analyzes financial model effectiveness and sustainability

## Always Include When
- Content focuses on comprehensive financial planning
- Financial modeling methodologies and frameworks
- Financial viability assessment approaches
- Long-term financial sustainability strategies

## Never Include When
- Basic accounting or bookkeeping content without strategic financial modeling

## Required Combinations
- Must include appropriate canvas-sections hierarchy: [economic-model, financial-model]

## Content Examples
- Financial modeling frameworks and methodologies
- Financial scenario planning and projections
- Financial planning and projection tools
- Financial viability validation and testing

## Related Tags
- Parent tags: [economic-model](../economic-model.md)
- Often appears with [revenue](../revenue/revenue.md) and [costs](../costs/costs.md) (components of financial model)
- May combine with venture-stage tags for stage-appropriate financial models

## Key Concepts
- Financial projections and scenarios
- Break-even analysis
- Cash flow management
- Financial sustainability metrics
- Social enterprise financial models
- Impact-revenue alignment
